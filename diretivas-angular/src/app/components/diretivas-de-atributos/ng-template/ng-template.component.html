• ngTemplate

    • O elemento do Angular ng-template define um template que não é renderizado por padrão.
    Observe que se você envolver o conteúdo dentro de um ng-template sem instruir o Angular para renderizá-lo,
    esse conteúdo não aparecerá em uma página.
    Documentação ngTemplate
