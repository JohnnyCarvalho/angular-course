• ngStyles

    • Uma diretiva de atributo que atualiza estilos para o elemento HTML contido. Define uma ou mais propriedades de estilo,
    especificadas como pares de valores-chave separados por dois pontos. A chave é um nome de estilo, com um sufixo opcional .
    unit(como 'top.px', 'font-style.em'). O valor é uma expressão a ser avaliada. O valor não nulo resultante, expresso na unidade
    fornecida, é atribuído à propriedade de estilo fornecida. Se o resultado da avaliação for nulo, o estilo correspondente será removido.
