    • {{ contador }}

    Decorador que marca um campo de classe como uma propriedade de entrada e fornece metadados de configuração.
    A propriedade de entrada está vinculada a uma propriedade DOM no modelo. Durante a detecção de alterações,
    o Angular atualiza automaticamente a propriedade de dados com o valor da propriedade DOM.
